# Meta Horizon Store Submission Checklist

Pre-submission checklist for JanetWorld.

## Developer Requirements

- [ ] Meta Developer account created
- [ ] Developer organization created
- [ ] App registered in Developer Dashboard

## App Assets

- [ ] App icon (1024x1024 and required sizes)
- [ ] Screenshots (landscape, Quest capture)
- [ ] Trailer / preview video (optional but recommended)
- [ ] Store listing copy (title, description, keywords)

## Permissions & Policy

- [ ] Privacy policy URL
- [ ] Permissions declared (Passthrough, tracking, etc.)
- [ ] Content rating completed

## Technical

- [ ] APK built in Release mode
- [ ] Signed with release keystore
- [ ] Tested on Quest 3 and Quest 3S
- [ ] No crashes on cold start
- [ ] Passthrough works as expected
